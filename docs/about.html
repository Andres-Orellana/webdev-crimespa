St. Paul Crime Map - Project 4

This single-page web app lets users explore recent St. Paul crime incidents by neighborhood. It includes map-based browsing, filters, neighborhood summaries, and incidents.

Tools & APIs Used

  • Vue: UI framework for reactive state, components, and rendering.
  • Foundation: responsive layout grid and UI styling.
  • Leaflet: interactive map, markers, popups, and bounds-limited navigation.
  • Nominatim: forward/reverse geocoding (address ↔ lat/lng).
  • Custom REST API (Project 3 server): SQLite-backed endpoints for crimes, codes, and neighborhoods.

Six Interesting Findings

  1. “Proactive Police Visit” shows up constantly across many neighborhoods (especially Capitol River)
  2. Some locations repeat, like Highland “20XX FORD PKWY” showing multiple theft reports close together
  3. A lot of incidents happen late night / early morning, with many timestamps between ~12:00am and 5:00am
  4. Violent/weapons incidents appear but seem less frequent than theft/proactive visit
  5. Property crime is common, especially Theft categories like shoplifting, theft from auto, and burglary
  6. Most incidents happen in Capital River in the center of St. Paul
